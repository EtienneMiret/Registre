Bonjour à tous ! Bienvenue sur Registre !

Vous pouvez ici enregistrer des références, puis les rechercher :

Les recherches les plus simples contiennent juste des mots-clés : hello world Dans ce cas, seules les références contenant tous les mots demandés remontent.

Vous pouvez utiliser l’opérateur or pour faire un OU logique entre deux mots : hello or world Dans ce cas, tous les termes de la recherche doivent être séparés par un or, hello or world war n’est pas une recherche valable.

Par contre, vous pouvez grouper vos termes avec des parenthèses : (a b) or c

L’opérateur type: permet de rechercher un type de référence particulier (film, bande-dessinée ou livre). Il prend en paramètre un mot-clé en anglais, movie, comic ou book : type:movie

Vous pouvez faire une recherche sur un champ particulier :

title:
Rechercher uniquement dans le titre d’une référence : title:home
comment:
Rechercher uniquement dans les commentaires d’une référence : comment:bonjour
series:
Rechercher uniquement dans les noms de séries : series:(big bang)
owner:
Rechercher uniquement parmis les propriétaires : owner:(Étienne)
location:
Rechercher uniquement parmis les emplacement : location:(Marcigny)
director:
Rechercher uniquement parmis les réalisateurs de films : director:(Steven Spielberg)
actor:
Rechercher uniquement parmis les acteurs de films : actor:(Anthony Head)
composer:
Rechercher uniquement parmis les compositeurs de musique de films : composer:(Howard Shore)
cartoonist:
Rechercher uniquement parmis les dessinateurs de bande-dessinées : cartoonist:(Hergé)
scriptWriter:
Rechercher uniquement parmis les scénaristes de bande-dessinées : scriptWriter:(Hergé)
author:
Rechercher uniquement parmis les auteurs de livres : author:(John Grisham)

Enfin, vous pouvez combiner tous ces opérateurs pour écrire des requêtes complexes.
